## Installing preparqation
- Install Java : https://www.java.com/en/download/manual.jsp
- Install JDK : https://www.oracle.com/middleeast/java/technologies/downloads/#jdk25-windows
- Set up java Path on windows
- Download intellij:https://www.jetbrains.com/idea/download
- Set Environment variable For Maven
- Download Repository from github
- open project on intellij
- download Cucamber for Java Plugin, So You can create Feature Files
- Run the runners class OR run each Feature File separately


##Note 
- in case SDK wasn't initialized please open project structure then select your SDK
